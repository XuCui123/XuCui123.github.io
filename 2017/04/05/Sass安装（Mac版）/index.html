Sass安装（Mac版）

1.确定是否安装了Ruby
ruby -v
2.安装方法
安装方法一：
安装Sass
使用sudo gem install sass进行Sass安装
安装方法二：本地安装
在该网站https://rubygems.org/gems/sass
下载Sass版本
使用gem install 文件地址
如gem install /Users/*/Downloads/sass-3.4.23.gem
3.使用sass -v 查看Sass版本号
sass -v
Sass 3.4.23 (Selective Steve)
